Bridging Visual and Auditory Domains: Object Recognition, Speech Synthesis with User Voice Input

This project integrates real-time object recognition with speech synthesis for enhanced accessibility. It combines YOLO for object detection with a Text-to-Speech engine to provide audio feedback for visually impaired users.

Tools Used: Python, YOLO, OpenCV, gTTS
